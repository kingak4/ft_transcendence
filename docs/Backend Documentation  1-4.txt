4.1 Interest-Based Chat Discovery
Users are automatically matched based on shared interests and skills.
Rules:
Users learning the same skill (e.g. Python) are grouped together
System generates dedicated chat spaces for each shared interest
Chats are dynamically suggested based on user activity and profile

4.2 Chat Creation
Users can create new chat rooms.
Requirements:
Chat must include a mandatory description
Example: “I am working on project X”
Content Restrictions:
Offensive language is strictly prohibited
Hate speech is strictly forbidden
Chat creation is blocked if:
Name contains prohibited content
Description contains prohibited content

4.3 Messaging Rules
Chat is text-only
Sending images, files, or any media is not allowed
Communication must remain purely textual

4.4 Social Interaction System (Friends Feature)
Users can:
Send friend requests to other users
Accept or reject invitations
Friendship Rules:
Private messaging is allowed only between friends
If users are not friends:
Private conversation is blocked
Blocking Behavior:
If a user removes a friend:
All private communication is automatically stopped
Access to private chat is revoked immediately

4.5 Chat Types
1. Interest-Based Chats
Automatically generated based on shared skills/interests
2. Public Group Chats
Users can create or join general-purpose groups.
Examples:
Support groups
Study groups
General discussion rooms

4.6 Moderation Rules
System validates:
Chat names
Chat descriptions
Messages
Any content containing:
Offensive language
Hate speech
must be blocked automatically.

4.7 Social Engagement & Achievements
The system includes social-based achievements.
Examples:
Adding friends increases progress toward achievements
Joining group chats contributes to engagement metrics
Active participation in chats is tracked

4.8 Invite System
Users can:
Invite friends to join the platform
Share a referral link
Features:
“Invite your friends” button available in the app
Rewards or achievements may be granted for invitations (to be defined)

4.9 Key Concept
The Community module ensures:
Safe communication environment
Learning-based social connections
Controlled private messaging (friend-only system)
Engagement tracking through social activity
